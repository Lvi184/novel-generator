## 🚀 快速开始

### 1. 添加新小说

```bash
cd case1

# 列出所有小说
python scripts/add_novel.py list

# 添加一本新小说
python scripts/add_novel.py "小说名称"

# 示例
python scripts/add_novel.py "诛仙"
```

这会自动创建完整的目录结构！

### 2. 上传小说文件

将小说 TXT 文件上传到对应的 `input/novel.txt`

例如：
- `data/zhu_xian/input/novel.txt`

### 3. 运行分析

修改配置文件中的小说路径，然后运行：

```bash
# Phase 1 - 深度解剖
python scripts/test_phase1_full.py

# Phase 2 - 骨架变形
python scripts/test_phase2.py

# Phase 3 - 逐章生成
python scripts/test_phase3.py
```

---

## 📝 注意事项

- 每本小说都有独立的文件夹，互不干扰
- 缓存是共享的，相同的 LLM 请求不会重复调用
- 添加小说时，脚本会自动生成拼音文件夹名
